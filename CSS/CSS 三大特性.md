# CSS 三大特性



## 层叠性

- 如果有样式冲突，**` 就近原则`**

## 继承性

1. 子标签继承父标签的一些样式

- 主要时文字相关的text ，font，line color

2. 行高1.5倍

```html
body{
font:12px/1.5
}

可以根据文字的大小自动调整行高
```
